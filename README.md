# 🏎️ Velocity X – F1 in Schools Website

## 🚀 Overview

**Velocity X** is an F1 in Schools team website built to showcase our engineering, team structure, and performance-focused car design.
The site reflects our core values: **precision, speed, and innovation**.

---

## 🌐 Features

* ⚡ Modern dark-themed UI inspired by Formula 1 aesthetics
* 📱 Fully responsive design (mobile + desktop)
* 🏎️ Inline embedded car model (no external assets required)
* 👥 Team section with defined roles
* 🤝 Sponsor showcase
* 📬 Contact form with interactive feedback
* 🎯 Smooth navigation and clean layout

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **Inline Base64 Image Embedding**

---

## 📁 Project Structure

```
velocity-x/
│── index.html   # Main website file (all-in-one)
│── README.md    # Project documentation
```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in any browser
3. That’s it — no setup, no dependencies
